WALT

We Are Learning To…

These learning intentions explain what you should understand and be able to do by the end of the Digital Inspection Bureau project.

DIGITAL LITERACY

Analyse Media Critically

  • Identify common techniques used to manipulate audiences online.
  • Question who benefits from a story and how it is presented.
  • Use evidence rather than emotion to judge reliability.
GAME DESIGN

Design Meaningful Systems

  • Translate real-world rules into clear game mechanics.
  • Create player choices with visible consequences.
  • Balance challenge, clarity, and fairness in gameplay.
PROGRAMMING

Code with Purpose

  • Create sprites, tilemaps, and game states in MakeCode Arcade.
  • Use variables and conditional logic to control outcomes.
  • Test, debug, and improve code through iteration.
COLLABORATION

Work Effectively with Others

  • Share ideas clearly and listen to different viewpoints.
  • Divide tasks responsibly within a team.
  • Give and receive constructive feedback.
SUCCESS

By the End of This Project

  • You can explain why a story may be manipulative.
  • You can show these ideas clearly inside a game.
  • You can justify your design choices using the rubric.
